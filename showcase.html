Showcase

Here are a few things I’ve built or achieved recently—everything from Java games to a machine-learning website, plus a snapshot of winning my high-school football state championship. Click on any preview to learn more!

Cool Cats: Intersectionality & Machine Learning

A web-based “journey” about intersectionality, built as a Teachable Machine demo. Users can drag-and-drop images to train an ML model that classifies pets. Under the hood: HTML/CSS/JS + Google’s Teachable Machine API.

View Live Site →

Froggie Feeding Frenzie (Java Game)

A simple desktop game written in Java where a frog character hops to eat diffent types of bugs without being hit. Built using Java w/ PApplet, custom rendering, and a health/score mechanic.

Cheddar’s Slot Machine Simulator (Java)

A fully functional slot-machine simulator coded in Java utilizing Javafx. Features: customizable bet amounts, reel animations, credit tracking, and a fun “business cat” theme starring Cheddar. Uses object-oriented principles (Reel, CreditManager, etc.) to manage game state.

Brookfield East: 2019 State Football Champions

As a Sophomore on the Brookfield East High School football team, I helped our Spartans win the Division 2 State Championship in 2019 at Camp Randall. That season we finished 12-2 and brought home the D2 WIAA trophy.

Highlight: 31-30 victory.
